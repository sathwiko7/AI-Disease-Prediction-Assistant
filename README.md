# 🩺 AI Powered Disease Prediction Assistant

An AI-powered healthcare application that predicts diseases based on user-selected symptoms using Machine Learning and provides disease descriptions, confidence scores, and recommended precautions.

## 🌐 Live Demo

🔗 **Try the Application:**
[https://ai-disease-prediction-assistant-kpi3azz4wph8chsefu3k5u.streamlit.app/]

---

## 🚀 Features

* Disease prediction using Random Forest Classifier
* User-friendly symptom selection
* Prediction confidence score
* Disease description display
* Recommended precautions
* Interactive Streamlit web interface
* Real-time predictions
* Clean and responsive UI

---

## 🛠 Tech Stack

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit

---

## 📊 Dataset

The model was trained using a healthcare symptom dataset containing:

* 130+ Symptoms
* 40+ Diseases
* Training and Testing datasets

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/sathwik07/AI-Disease-Prediction-Assistant.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
